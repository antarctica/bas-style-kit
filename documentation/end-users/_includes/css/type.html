Body copy

A global default font-size of 14px, with a line-height of 1.428 is used. This is applied to the <body> and all paragraphs. In addition, <p> (paragraphs) receive a bottom margin of half their computed line-height (10px by default).

Built with Less

The typographic scale is based on two Less variables, @font-size-base the base font-size used globally, and @line-height-base, the base line-height. These variables and simple maths are used to create the margins, paddings, and line-heights of all our typographic styles and more. The BAS Style Kit does not override these variables and uses the Bootstrap defaults. If you wish to customise them yourself, all styles will adjust automatically, providing you override these variables.

BAS Typography

The BAS Style Kit includes two additional fonts, Open Sans and Gill Sans, which are applied to all elements and components as described below.
In general, Open Sans is used as a base font applied everywhere by default, with Gill Sans used as a brand font for specific, targeted elements and components.

Why two fonts?

Gill Sans is the font used in the BAS logo and is used as the corporate, or brand font. It is used in specific areas, usually high impact areas (such as headings) as a design feature. There is no technical reason for using Gill Sans - however it should be used accordance with the BAS Style Guide.

Domain white-listing for web-font usage

In order to use this web-font within a website or application its domain must be white-listed within the BAS font subscription. The following common domains are allowed:

  • *.bas.ac.uk

There is no limit to the number of domains that can be white-listed and wild cards are supported and additional domains can be added by submitting feedback.

Note: Only general/common domains are listed here. Development environments will also need to white-listed and will count towards our page-view limit.
