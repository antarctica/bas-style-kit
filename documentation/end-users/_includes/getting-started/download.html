Download

There are currently two ways to download the BAS Style Kit (currently v{{ site.current_version }}) depending on your use case.

Compiled styles (i.e. CSS)

The simplest and quickest way to integrate the Syle Kit, suits single pages and small/medium websites/web-apps. Use this version if you do not intend to modify the styles of the Style Kit significantly and/or you wish to take advantage of the BAS CDN.

Includes both regular (bas-style-kit.*) and minified (bas-style-kit.min.*) compiled CSS. CSS source maps (bas-style-kit.*.map) are available for use with certain browsers' developer tools. Web-fonts for Open-Sans and iconic fonts are also included.

Source styles (i.e. Less)

For more advanced use ages where significant modifications are neded, or you wish to integrate the Style Kit in larger or complex projects.

This version includes pre-compiled CSS and web-fonts, along with source Less styles and documentation.
Use this version if you plan to integrate the Style Kit into a larger application with its own styling workflow. Requires a Less compiler and some setup.

BAS Style Kit CDN

If using the compiled styles, you can make use of the BAS CDN to speed up loading times between BAS websites and apps.

This works because the same files, from the same URL (i.e. the CDN) are used by each website or app and cached by the browser.

Due to the performance benefits and lack of need to include in your projects directly, it is recommended to use the BAS CDN wherever practical.
