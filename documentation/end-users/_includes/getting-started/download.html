Install with npm

You can also install the BAS Style Kit using npm:

{% highlight bash %}$ npm install bas-style-kit{% endhighlight %}

The BAS Style Kit's package.json contains some additional metadata under the following keys:

  • less - path to the main Less source file
  • style - path to the pre-compiled, but non-minified, CSS file

The BAS Style Kit module depends on the Bootstrap module, callrequire('bootstrap') will load all of Bootstrap's jQuery plugins onto the jQuery object. The bootstrap module itself does not export anything. You can manually load Bootstrap's jQuery plugins individually by loading the /js/*.js files under the package's top-level directory.

Autoprefixer required for Less

The BAS Style Kit uses Autoprefixer to deal with CSS vendor prefixes. If you're compiling the BAS Style Kit from its Less source and not using our Gruntfile, you'll need to integrate Autoprefixer into your build process yourself. If you're using the pre-compiled CSS or using our Gruntfile, you don't need to worry about this because Autoprefixer is already integrated into our Gruntfile.
