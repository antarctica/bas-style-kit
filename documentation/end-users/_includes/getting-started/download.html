Download

There are currently two ways to download the BAS Style Kit (currently v{{ site.provisioned.version }}) depending on your use case.

Compiled styles (i.e. CSS)

  • The simplest and quickest way to integrate the Style Kit, suits single pages and small/medium websites/web-apps.
  • Includes both regular and minified compiled CSS, plus CSS source maps and fonts for Open-Sans and iconic web-fonts.
  • Use this version if you do not intend to modify the styles of the Style Kit significantly and/or you wish to take advantage of the BAS CDN.

Source styles (i.e. Less)

  • For more advanced use ages where significant modifications are neded, or you wish to integrate the Style Kit in larger or complex projects.
  • Includes everything from the Compiled Styles, along with source Less styles, this documentation and a Vagrant and Ansible based development environment.
  • Use this version if you plan to integrate the Style Kit into a larger application with its own styling workflow. Requires a Less compiler and some setup.

BAS CDN

When using the compiled styles, you should use the BAS CDN to speed up loading times between BAS websites and apps.

This works because the same files, from the same URL (i.e. the CDN) are used by each website or app and cached by the browser.
