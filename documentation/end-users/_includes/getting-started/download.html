Download

There are two ways to download the BAS Style Kit (currently version {{ site.version }}):

Compiled styles (i.e. CSS)

  • Simplest and quickest way to integrate the Style Kit, suits websites and simple web-apps.
  • Includes both regular and minified compiled CSS, plus CSS source maps and typographic and iconic web-fonts.
  • Use this version unless you need know you use another, and/or you wish to take advantage of the BAS CDN.

Source styles (i.e. Less)

  • Where you want to build the Style Kit into larger projects, or only use specific elements.
  • Includes everything from the Compiled Styles plus; Less source styles, this documentation and the development environment.
  • Requires a Less compiler and additional setup.
