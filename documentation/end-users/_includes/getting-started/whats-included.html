What's included

The BAS Style Kit is available in two forms, within which you'll find the following directories and files, logically grouping common resources and providing both compiled and minified variations.

jQuery required

Note: Bootstrap, on which this project depends, requires jQuery for all of its JavaScript plugins. Therefore you must include jQuery before Bootstrap, and then the BAS Style Kit JavaScript. See the starter template for an example.

This is the most basic form of integrating the BAS Style Kit: pre-compiled files for quick drop-in usage in nearly any web project. We provide both compiled CSS and JS (bas-style-kit.*), as well as compiled and minified CSS and JS (bas-style-kit.min.*). CSS source maps (bas-style-kit.*.map) are available for use with certain browsers' developer tools. Web-fonts for Open-Sans are also included.

The less/ and js/ directories are the source code for our CSS and JS respectively. The dist/ folder includes everything listed in the pre-compiled version detailed above. The docs/ folder includes the source code for our documentation, and examples/ of the BAS Style Kit's usage. Beyond that, any other included file provides support for packages, license information, and development.
