Footer

Footers are a containing component that can hold anything you want, such as additional navigation, legal and contact information or anything else you want.

This component is unique to the BAS Style Kit, but modelled on the navbar component.

Default footer

Footers can range from very simple, minimal elements containing simple text, to large fat footers containing lots of links, information and actions.
The footer component is designed to be compatible with almost all elements and components to allow you to use it as your project demands.

Default footer content

The BAS Style Guide details some required and recommended content to include in footers.

Body padding required

The fixed footer will overlay your other content, unless you add padding to the bottom of the <body>.

Make sure to include this after the core Bootstrap CSS & BAS Style Kit CSS.

Static bottom

Create a full-width footer that scrolls away with the page by adding .footer-bsk-static-bottom and include a .container or .container-fluid to center and pad footer content.

Unlike the .footer-bsk-fixed-bottom class, you do not need to change any padding on the body.
