Image purchase

The image purchase component is used to identify images available for purchase from the BAS Image Collection.

This component is unique to the BAS Style Kit and designed to cross promote images through our projects and other outputs.

Prototype component

This is a new component and very much a 'work in progress'. If you spot any layout or other bugs please let us know.

We have plans to improve how this component works so it won't be necessary to manually insert any markup, other than adding a data-* attribute.

Image purchase symbol

The image purchase symbol is an icon, currently (.fa .fa-camera) overlain on relevant images.
The symbol is used to identify images as being available for purchase, and to provide a direct link to do so.

Supported components

The image purchase symbol is currently only supported when used within the Thumbnail or Jumbotron components, and then only when using the BAS Style Kit variants.
If you need support for another component, or you've done testing yourself using another component please let us know or submit a pull request.

Thumbnails

Alternative markup required for thumbnails without a caption

Where a thumbnail has no caption (i.e. it's just an image) the class .bsk-image-purchase-no-caption must be applied.

Alternative markup required for linked thumbnails

Where a thumbnail is a link (i.e. a a.thumbnail rather than a div.thumbnail) the image purchase symbol cannot be added as a link (as this would be a link within a link).
To overcome this JavaScript is used to make the symbol function like a link, and CSS to make the symbol look like a link. We hope to improve this in future by auto-generating the purchase link based on the parent element.
