BAS jumbotron

The BAS Style Kit includes two styles of jumbotron, default and inverse, designed to match the BAS design.

Default BAS jumbotron

Add .jumbotron-bsk with .jumbotron-bsk-default to produce a default jumbotron using BAS styling.

You should always use a jumbotron variant

As with buttons and other components you should always use a jumbotron variant, such as .jumbotron-bsk-default or .jumbotron-bsk-inverse rather than just the base class.
