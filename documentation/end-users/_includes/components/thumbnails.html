Thumbnails

Extend the grid system with a thumbnail component to easily display grids of images, videos, text, and more.

If you're looking for Pinterest-like presentation of thumbnails of varying heights and/or widths, you'll need to use a third-party plugin such as Masonry, Isotope, or Salvattore.

The BAS Style Kit includes additional thumbnail variants designed to match the BAS brand.

BAS thumbnails

The BAS Style Kit includes two styles of thumbnail, default and inverse, designed to match the BAS design.

Default BAS thumbnail

Add .thumbnail-bsk with .thumbnail-bsk-default to produce a default thumbnail using BAS styling.

You should always use a thumbnail variant

As with buttons and other components you should always use a band variant, such as .thumbnail-bsk-default or .thumbnail-bsk-inverse rather than just the base class.
