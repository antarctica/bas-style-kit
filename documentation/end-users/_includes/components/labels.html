Available variations

Use any of the standard modifier classes to change the appearance/context of a label.

Make sure label contexts are accessible

If using a label to convey some sort of context, an error for example, ensure you include .sr-only text to convey this to users of assistive technologies. This applies to all contexts, including success, info, etc.

Have tons of labels?

Rendering problems can arise when you have dozens of inline labels within a narrow container, each containing its own inline-block element (like an icon). The way around this is setting display: inline-block;.
