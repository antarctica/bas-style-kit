Header

Headers are a simplified version of a navbar for use in external systems.

This component is unique to the BAS Style Kit, where the appearance of a navbar is desired, without the complex markup.

Default header

You should probably use a navbar

Headers are only really designed to give the illusion of a navbar in external systems.
It is not designed for use in projects based on this Style Kit, use the navbar component instead.

Using images with headers

Headers need an additional class, .header-bsk-img-logo, applied directly to the img element
to fix its padding. Without this class too much padding will be applied. An example using the BAS logo is shown below.
