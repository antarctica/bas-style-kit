Mixed: mobile and desktop

The BAS Style Kit adds an additional grid size to give five tiers of classes: xs (phones), sm (tablets), md (tablets, small desktops), lg (desktops) and xl (large desktops - custom to the BAS Style Kit). You can use nearly any combination of these classes to create more dynamic and flexible layouts.

Each tier of classes scales up, meaning if you plan on setting the same widths for xs and sm, you only need to specify xs.
